Основные требования: 
1. Верстка PixelPerfect. Адаптивность, кроссбраузерность.
2. Окно вывести на странице, по нажатию на кнопку.
3. Использование bootstrap разрешено, по желанию.
4. При верстке обязательно использование scss, BEM (нейминг)
5.  Фронтенд должен собираться через webpack 
6. Минимальные требования: статика с использование Vue компонента (как минимум одного, достаточно внедрить верстку в его темплейт). Если позволяет уровень - добавить динамики, детальнее в комментах ниже.
